# Vanilla `renderTags` example (Wave 4)

Runnable **FEATURES D7** sample: import **`@better-seo/core`**, build an **`SEO`** document with **`createSEO`**, then print **`renderTags(seo)`** for non-React hosts or snapshots.

From the monorepo root (after `npm install`):

```bash
npm run demo -w vanilla-render-tags-example
```

(`-w` uses the package **name** from this folder’s `package.json`.)

Or from this directory:

```bash
npm install
npm run demo
```

No bundler required — **ESM** + Node 20+. See **[`docs/recipes/README.md`](../../docs/recipes/README.md)** and **`@better-seo/core`** `renderTags` / `TagDescriptor` in the package source.
